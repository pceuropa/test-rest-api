# Test-rest-api
A command-line tool based on python3 and pytest for testing of RESTful APIs. 


REQUIREMENTS
------------
Python 3.6+
pytest
inotifywait - to automatic tests


Instalation
---------------
```
pip3 install -U pytest
```

Tests manual
---------------
```
pytest -v tests.py
```

Tests automatic
---------------
Runner run test every saves
```
./test_runner.sh
```
